# DimeTrack

Local-first revenue + expense tracker (Node + Express + SQLite).

## Run

1. Install deps

```bash
npm install
```

2. Start

```bash
npm start
```

Then open http://localhost:3000

## Notes

- Data lives in `data.db` in the project folder.
- Recurring items: add rent/subscriptions/paycheck, then click **Post** to create a transaction and move `next_due_date` forward.
